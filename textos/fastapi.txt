✅ Regras ao criar uma API com FastAPI (aplicadas ao seu sistema)
Autenticação obrigatória → endpoints devem exigir login (via JWT ou sessão).

Autorização → cada usuário só acessa seus próprios carrinhos/itens.

Isolamento → não retornar dados de outros usuários.

Validação → usar pydantic para validar entrada (ex: nome do carrinho, item).

Respostas claras → sempre retornar JSON estruturado (sucesso/erro).

Segurança → nunca expor senha, apenas hash (já está no seu models.py).

Documentação automática → usar /docs para testar endpoints.
