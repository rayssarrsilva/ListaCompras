1️⃣ Parar tudo
docker compose down

2️⃣ Reconstruir tudo (obrigatório!)
docker compose up --build

down -v apaga volumes (limpa o banco antes de recriar)
docker compose down -v

Código → up --build
Configuração/env → down + up
Dados/volumes → down -v + up --build

docker compose down -v
docker compose up --build

Se notar comportamento estranho, rode (Limpa containers parados, imagens não usadas e cache): docker system prune -af

→ Lista containers ativos e ajuda a confirmar se tudo está rodando.
docker ps

→ Útil se você quiser reiniciar depois sem recriar.
docker compose stop
docker compose start

Parar um container específico:
docker stop nome_do_container
